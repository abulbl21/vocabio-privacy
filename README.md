# Vocabio Gizlilik Politikası

Son güncelleme: Nisan 2026

Bu gizlilik politikası, Vocabio uygulamasının kullanıcı verilerini nasıl topladığını, kullandığını ve koruduğunu açıklamaktadır.

## Toplanan Veriler

Vocabio, kullanıcıların eklediği kelimeler ve projeler cihazda yerel olarak saklanır. Uygulama, haber içeriklerini görüntülemek için Firebase Firestore hizmetini kullanmaktadır. Kullanıcı hesabı oluşturulmaz ve kişisel bilgi toplanmaz.

## Üçüncü Taraf Hizmetler

Uygulama aşağıdaki üçüncü taraf hizmetleri kullanmaktadır:
- Google Firebase (içerik sunumu için)
- OpenAI API (kelime çevirisi için)

Bu hizmetler kendi gizlilik politikalarına tabidir.

## Veri Güvenliği

Kullanıcı verileri yalnızca cihazda saklanır ve üçüncü taraflarla paylaşılmaz.
